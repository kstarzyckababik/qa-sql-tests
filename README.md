# SQL for QA - E-commerce examples

## Project description
A collection of SQL queries used for testing and validating e-commerce data.
The project focuses on typical QA scenarios.

## Database structure
- users
- products
- orders
- order_items

## SQL topics covered
- SELECT
- WHERE
- JOIN
- GROUP BY
- Aggregations

## Use cases
- Verify order totals
- Validate user orders
- Check product data consistency
